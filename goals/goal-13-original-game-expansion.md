# Goal 13 — Original Game Expansion

## Mission

Build BoardLink-exclusive original games beyond Bubble Siege.

This goal covers:

- Tap Territory
- Color Catcher
- Memory Raid
- Sync Bomb
- Path Block (optional, if the first four are stable)

## Recommended implementation order

1. Tap Territory
2. Color Catcher
3. Memory Raid
4. Sync Bomb
5. Path Block

## Required outcomes

### Tap Territory

- square or hex logical board
- simultaneous claim handling
- 2-player version first
- timed round
- result count
- replay

### Color Catcher

- target-color assignment
- mixed object display
- correct-selection scoring
- incorrect-selection penalty
- role switch
- replay

### Memory Raid

- reveal phase
- hidden phase
- target prompt
- remembered-position interaction
- result
- replay

### Sync Bomb

- player-specific hint projection
- cooperative timer
- sequence or puzzle resolution
- success/failure result
- replay

### Path Block

- attacker path drawing
- defender blocker placement
- collision rule
- time comparison or clear round result
- replay
