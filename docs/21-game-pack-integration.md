# Game Pack Integration Plan

## Shared systems reused by all added games

Before implementing the following packs, confirm the platform already provides:

- room creation
- room invitation
- ready state
- deterministic game registry
- same-device local authority
- replay support
- result screen
- local history
- reconnect
- accessibility labels
- localization strings
