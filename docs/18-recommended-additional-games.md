# Recommended Additional Games

## 1. Four in a Row

- 2 players
- 2–5 minutes
- low complexity
- fixed board
- drop animation
- four-in-line detection
- replay
- same-device mode

## 2. Reversi

- 2 players
- legal move generation
- forced pass
- flip resolution
- final scoring
- replay
- same-device mode

## 3. Dots and Boxes

- 2–4 players
- line claim
- box detection
- extra-turn rule
- final scoring
- replay
- same-device mode

## 4. Number Baseball

- 1v1 hidden number mode
- 3-digit / 4-digit options
- strike / ball evaluation
- player-specific projection
- rematch
- same-device mode

## 5. Rock-Paper-Scissors Tournament

- 2–64 players
- simultaneous hidden choice
- reveal step
- tie handling
- bracket mode
- result summary

## 6. Quiz Buzzer

- 2–32 players
- host view
- participant buzzer view
- first-buzz lockout
- reset
- scoreboard

## 7. Yutnori

- 1v1 and 2v2
- fixed rule profile
- server-authoritative stick result
- move options
- team result
- replay

## 8. Fleet Grid

- 2 players
- ship placement
- hidden board projection
- hit/miss/sunk result
- replay
- same-device mode

## 9. Secret Role

- 3–12 players
- hidden role or hidden word distribution
- timed phase
- vote phase
- reveal
- curated content pack only

## 10. Majority Prediction

- 3–20 players
- curated prompt packs
- majority mode
- minority mode
- round scoring
- host reset
