# Original BoardLink Game Ideas

## 1. Tap Territory

Players race to claim cells on a shared board. The server grants ownership to the earliest valid claim.

## 2. Color Catcher

The attacker chooses a target color. The defender must quickly pop only balls of that color.

## 3. Memory Raid

Players briefly study a layout, then must remember and act on it after it disappears.

## 4. Sync Bomb

Players each see only part of the information needed to solve a timed puzzle.

## 5. Path Block

One player tries to draw or navigate a route while the opponent places temporary blockers.

## Original-game design rule

Every original game should satisfy at least 4 of these 6:

1. understandable in under 20 seconds
2. playable in under 2 minutes
3. visually interesting for spectators
4. input-light for mobile
5. easy to explain by invitation
6. unique enough to represent BoardLink
